# Expensify app

Simple react redux CRUD app.

## How to run

Clone the repo, cd into folder and run this commands.

```
npm install
npm run build:prod
npm start
```

Open your browser and go to this address:

http://localhost:3000/

## How to run in development mode

```
npm run dev-server
```

Open your browser and go to this address:

http://localhost:8080/
